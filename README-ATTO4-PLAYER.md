# Atto4 Player - Self-Hosted HLS Streaming System

## ⚖️ Legal Compliance

**IMPORTANT**: This player accepts and plays only sources you are licensed to use. 

- Do NOT use this system to bypass provider protections
- Do NOT scrape copyrighted streams without permission
- Do NOT extract content you don't own or have rights to
- Only add sources where you have proper licensing agreements

This software is provided for legitimate streaming of owned or licensed content only.

## Features

✅ Custom HLS player with hls.js  
✅ M3U8 catalog management with fallback sources  
✅ Streamlink integration for stream extraction  
✅ Watch progress & telemetry tracking  
✅ Continue watching functionality  
✅ Quality selector with adaptive streaming  
✅ Picture-in-picture support  
✅ Fullscreen mode  
✅ Keyboard shortcuts  
✅ Dual-mode: Custom player OR third-party iframe
